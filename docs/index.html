BetterPR

BetterPR is a simple CLI tool to improve the UX of Bitbucket PRs. Instead of having multiple tabs open, or clicking between projects, BetterPR allows you to get information about open PRs in one cohesive view across multiple projects in your workspace.
Installation is as easy as running the following command:
npm i -g @hunterparks/betterpr
For more information, please see the project's Github page.
