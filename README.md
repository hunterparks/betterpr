# BetterPR

BetterPR is a better way to look at PRs in Bitbucket!

## Installation

Use the package manager [npm](https://npmjs.com/) to install BetterPR.

```bash
npm install --global @hunterparks/betterpr
# or, even shorter
npm i -g @hunterparks/betterpr
```

## Usage

```bash
betterpr
```
